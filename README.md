# SmartNotes AI

A full-stack note-taking app with AI chat.
Write notes, upload PDFs, then ask questions
about your own content using an LLM.

**Live demo:** [coming soon]

## Tech stack
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Auth: JWT
- AI: Anthropic Claude API

## Features
- JWT authentication (register + login)
- Create, edit, delete notes
- Upload PDFs — text is extracted + stored
- AI chat: ask questions about any note or PDF
